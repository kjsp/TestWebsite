Usecase Board

유스케이스 작성 방법
1. 반드시 주어(실행자)을 넣는다.
2. 한번에 실행되는 단위로 글목록을 구분한다.
3. 예외는 소문자로 표기한다.
4. 반드시 사용자 입장에서 작성한다.

게시판 글 목록 보기
1. 사용자는 게시판 보기를 누른다.
2. 시스템은 DB에 접근해 글 목록을 가져와 출력한다.(board.jsp)
	 2.a 글이 존재하지 않을시 글 등록 부탁 메세지를 출력한다.
3. 사용자는 페이징을 통해 글 목록을 탐색할 수 있다.
4. 시스템은 ajax로 2부터를 반복한다.

게시판 글 보기
1. 사용자는 글 목록 보기에서 글 제목을 누른다.
2.시스템은 DB에 접근해 해당 글 정보를 가져와 출력한다.(boardView.jsp)
3.사용자는 글 정보를 볼 수 있다.

게시판 글 쓰기
1. 사용자는 게시판 글 목록 보기에서 글쓰기 버튼을 누른다.
2. 시스템은 postBoard.jsp를 가져와 출력한다.
3. 사용자는 글 작성을 한다.
	3.a 사용자가 제목이나 글내용을 입력하지 않으면 제목/내용을 입력해주세요 메세지를 출력
4. 시스템은 작성된 글을 db에 저장한다.

게시판 글 수정
1. 사용자는 게시판 글 보기에서 수정 버튼을 누른다.
2. 시스템은 updateBoard.jsp 를 가져와 제목, 내용 부분을 활성화 시켜 출력한다.
3. 사용자는 글을 수정한다.
	3.a 사용자가 제목이나 내용을 지우고 수정완료 버튼을 눌렀을 때 제목/내용을 입력해주세요 메세지를 출력
4.시스템은 db에 수정된 글을 저장한다.

게시판 글 삭제
1. 사용자는 게시판 글 보기에서 삭제 버튼을 누른다.(deleteBoard.jsp)
2. 시스템은 삭제하시겠습니까 라는 confirm 을 띄운다.
3. 사용자는 확인버튼을 눌러 게시글을 삭제한다.
4.시스템은 db에서 해당 글을 삭제한다.
