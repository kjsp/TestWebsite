

회원가입
1. 사용자는 index.jsp에서 회원가입 버튼을 누른다.
2. 시스템은 회원 가입 페이지로 이동한다.
3. 사용자는 ID, password, nickname, e-mail을 입력하고 회원가입을 누른다.
  3.1 시스템은 javascript를 이용하여 필수 조건을 체크한다.
  3.2 시스템은 ajax를 통하여 ID의 중복을 체크한다.
  4. 시스템은 DB에 정보를 저장하고 메세지를 출력한다.
  
  내정보 보기
  1.사용자는 main.jsp에서 '내정보'를 버튼을 누른다.
  2.시스템은 DB에 접근하여 'ID'에 해당하는 사용자의 정보를 내정보 보기 페이지에 표시한다.
  3시스템은 내정보 보기 페이지에서 '수정' , '회원탈퇴' 버튼을 제공한다.
  
  
  탈퇴
  1.사용자는 내정보 페이지에서 회원 탈퇴 버튼을 누른다.
  2.시스템은 비밀번호를 확인하는 페이지로 이동한다.
  3. 사용자는 비밀번호를 입력한다.
  4. 시스템은 비밀번호를 비교한다.
  	4.1 시스템은 사용자가 입력한 비밀번호를 DB에 저장된 정보와 비교하여 일치 시 삭제한다.
  	4.2 시스템은 불일치 alert를통해 사용자에게 알려준다.		
 5.  시스템은 index.jsp 이동한다
  		
  수정
  1.사용자는 내정보 페이지에서 수정 버튼을 누른다
  2.시스템은 비밀번호 확인 페이지로 이동한다.
  3.사용자는 비밀번호를 입력한다.
  4.시스템은 비밀번호를 확인한다.
  		4.1 시스템은 사용자가 입력한 비밀번호를 DB에 저장된 정보와 비교하여 일치 시 수정페이지로 이동한다.
    	4.2 시스템은 불일치 alert를통해 사용자에게 알려준다.	
  5. 사용자는 수정 하고자 하는 내용을 수정 후 수정 버튼을 누른다
  		5.1 시스템은 id를 수정 하지못하게 비활성화 시킨다.
  		5.2 시스템은 javascript 를 통해 수정되는 내용중 공란이 있는지 체크한다.
  6.시스템은 main.jsp  페이지로 이동한다.
